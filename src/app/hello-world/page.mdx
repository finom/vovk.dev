# "Hello World" showcase

The "Hello World" app is a simple example that showcases some of the core features of Vovk.ts. It's made of:

- `UserController` and `UserService` that implement a dummy endpoint as `updateUser` JSON handler.
- `StreamController` and `StreamService` that implement a dummy endpoint as `streamTokens` [JSONLines](/controller/jsonlines) handler.
