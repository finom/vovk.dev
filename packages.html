Packages of the Main MonorepoÂ :

vovkÂ 

The core/runtime library providing decorators, utilities, types, and other features used by both server-side and client-side code. It peer-depends only on type-focused openapi3-tsÂ  and is reported as 100% self-composed on BundlephobiaÂ .

npm install vovk
pnpm add vovk
yarn add vovk
bun add vovk

vovk-cliÂ 

The CLI package. Install globally or as a dev dependency. Provides the vovk binary.

npm install vovk-cli --save-dev
pnpm add vovk-cli --save-dev
yarn add vovk-cli --dev
bun add vovk-cli --dev
npx vovk-cli <command>
pnpm dlx vovk-cli <command>
yarn dlx vovk-cli <command>
bun x vovk-cli <command>

Or:

npx vovk <command>
pnpm dlx vovk <command>
yarn dlx vovk <command>
bun x vovk <command>

vovk-clientÂ 

The client package that re-exports the composed TypeScript client, typically located in node_modules/.vovk-client. Import it from client-side code (Node.js or browser) to access RPC methods and schemas.

npm install vovk-client
pnpm add vovk-client
yarn add vovk-client
bun add vovk-client

vovk-ajvÂ 

Exposes validateOnClient to validate input data on the client using the emitted JSON Schema.

npm install vovk-ajv
pnpm add vovk-ajv
yarn add vovk-ajv
bun add vovk-ajv
