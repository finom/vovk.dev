  • An OpenAPI specification that describes the API endpoints powered by Scalarâ€‰.
  • Building and packaging the app with Vovk CLI and external tools.

Bundled and generated files are available in the Git index as distâ€‰, dist_rustâ€‰ and dist_pythonâ€‰, so you can explore (usually they should be .gitignoreâ€™d). 31          })
32          .meta({ description: "User profile object" }),
33      })
34      .meta({ description: "User data object" }),

36      .object({

42      .meta({

46      .object({

53      }),

55      .object({

64    },
65  });
66}
