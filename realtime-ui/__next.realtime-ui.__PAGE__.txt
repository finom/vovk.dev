# Realtime Kanban — AI-native UI updates with Vovk.ts

**Realtime UI** is a streaming-first architecture for Next.js that keeps the UI in perfect sync with the back-end using only JSON Lines and Vovk.ts.
This series walks through **Realtime Kanban** — the reference app that implements it — showing how users, bots, AI agents, and MCP clients can all update the board in real time with almost zero extra code.

<DocIndexCards meta={meta} icons={icons} hrefPrefix="realtime-ui" />


## See it in action

### AI agent managing the board via MCP

Claude connects to the Kanban board through an [MCP server](/realtime-ui/mcp) and creates, moves, and deletes cards autonomously.

<video src="/video/kanban_mcp.mp4" type="video/mp4" className='mt-4' controls loop />

### Multi-user collaboration with live polling

Multiple users edit the same board simultaneously — changes propagate in real time through [database polling](/realtime-ui/polling) and [normalized state](/realtime-ui/state).

<video src="/video/kanban_polling.mp4" type="video/mp4" className='mt-4' controls loop />

### Chat-driven board updates with function calling

A built-in [text chat interface](/realtime-ui/text-ai) lets users manage cards through natural language, powered by OpenAI function calling.
